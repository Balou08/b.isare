Level 0

Bienvenue dans B.ISARE votre ojectif est de trouver le mot du niveau suivant. Utilisez b.isare0 comme username et isare-ctf.helptech.space comme adresse du serveur

commande complete pour la connexion ssh


ssh -l b.isare0 isare-ctf.helptech.space -p 26370 ou ssh b.isare0@isare-ctf.helptech.space -p 26370

le mot de passe se trouve dans un fichier README dans le repertoire courant de l'utilisateur b.isare0.les commandes qui pourront etre utile

ls, cd, cat, less, more, head, pwd.

Level 1

Felicitations pour votre premiere victoire dans B.ISARE.Pour le niveau suivant utilisez b.isare1 comme username et isare-ctf.helptech.space -p 26370 comme adresse du serveur


commande complete pour la connexion ssh

Le mot de passe se trouve dans l'un des fichiers tirets du repertoire courant de l'utilisateur.les commandes qui pourront etre utile

Level 2

Le mot de passe se trouve dans un ficher contenu dans l'un des sous-repertoires de l'unique repertoire present dans le repertoire courant de l'utilisateur et possede les caracteristiques suivantes:
le proprietaire est root
il fait partie du groupe b.isare2
fait 4.0ko

commande complete pour la connexion ssh


ssh -l b.isare2 isare-ctf.helptech.space -p 26370 ou ssh b.isare2@isare-ctf.helptech.space -p 26370

les commandes qui pourront etre utile

ls, cd, cat, less, more, head, pwd, locate, find, grep, du, man.

Level 3

Le mot de passe est divise en deux parties l'une se trouvant dans le permier repertoire present dans le repertoire courant de l'utilisateur et l'autre partie dans le second repertoire. Creez un repertoire temporaire de travaille.trouvez les fichiers contenant les mots de passe et ensemblez les dans le bon ordre. N'oubliez pas :

- Dans le premier fichier le mot de passe est la seule chaine de caractere contenat le caractere "6"
- Dans le premier fichier la 1er partie du mot de passe est la seule chaine de caractere qui contient le caractere "*" et est la 3eme des 10 dernieres lignes du fichier

commande complete pour la connexion ssh


ssh -l b.isare3 isare-ctf.helptech.space -p 26370 ou ssh b.isare3@isare-ctf.helptech.space -p 26370

les commandes qui pourront etre utile

ls, cd, cat, less, more, head, pwd, find, du, grep,tac, cat, tail, head,
mktemp, xxd, tar, xz, base64, hexdump, tr, uniq, man, cp, mv, sort.

Level 4

Le repertoire courant de cet utilisateur ne contient aucun fichier contenant le mot de passe du niveau suivant mais une clef prive pour une connexion ssh pour acceder au niveau suivant

les commandes qui pourront etre utile

ls, cd, cat, less, more, head, pwd, cp, mv, ftp, scp, ssh.
